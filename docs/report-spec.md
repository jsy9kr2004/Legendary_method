# 레포트 명세 (report-spec.md)

종배 매매 의사결정을 보조하는 레포트 포맷 정의.

## 레포트 종류

| 시점 | 종류 | 채널 | 길이 |
|---|---|---|---|
| 09:30 | 모닝 정기 | 텔레그램 | 중간 |
| 11:00 | 1차 추적 | 텔레그램 | 짧음 |
| 13:00 | 2차 추적 | 텔레그램 | 짧음 |
| 14:00 | 3차 추적 | 텔레그램 | 짧음 |
| **14:50** | **결정** ★ | **텔레그램** | **상세** |
| 상한가 진입 | 이벤트 트리거 ★ | 텔레그램 푸시 | 매우 짧음 |
| 16:00 | 사후 정기 | 이메일 | 매우 상세 |

---

## 1. 모닝 정기 레포트 (09:30)

목적: 시장 국면 판단 + 어제 매수 종목 갭 결과 확인.

```markdown
📊 [모닝] 2025-05-05 (월)
═══════════════════════════════

[시장 국면 지표]
KOSPI 시초:    2,XXX.XX  (전일 +X.XX%)
200일 이평:    2,XXX.XX  → 시초 위/아래
60일 수익률:   +X.XX%
VKOSPI:        XX.XX
20일 음봉비율: XX%

야간 미증시: S&P +X.XX% / NASDAQ +X.XX%
시간외 단일가 흐름: 강세/약세 (종목 수 기준)

[보유 종목 갭 결과]  ← Zeta 본인 매수 종목 (수동 입력)
제룡전기  매수가 91,300 → 시가 9X,XXX (+X.X%)
                          → 권장: 시초 익절

→ 대세상승장 판단: [Zeta 직관]
```

## 2. 정기 추적 레포트 (11:00 / 13:00 / 14:00)

목적: 주도테마 변화 감지 + 신규 상한가 알림.

```markdown
📊 [추적-11:00] 2025-05-05
═══════════════════════════════

[거래대금 30위 — 테마 분포]
1위 SK하이닉스           반도체
2위 삼성전자             반도체
3위 LS ELECTRIC          전력기기/원전
4위 대원전선             전선/전력
...

[주도테마 (≥3종목)]
• 반도체 (5)         : 하닉, 삼전, ...
• 전력기기 (4)       : LS, 제룡전기, ...
• AI 데이터센터 (3)  : ...

[신규 상한가] (직전 추적 이후)
• 제룡전기 (전력기기)  11:23 도달
  → 상세는 결정 레포트(14:50)에서

[테마 변화]
• 신규 진입: 전력기기
• 약화: (없음)
```

## 4. 상한가 진입 이벤트 알림 ★

목적: 박민준의 "91,300원에 상한가 칠 때 들이밀었다" 패턴 포착.

```markdown
🚨 [상한가] 14:23
제룡전기 (075180)
주도테마: 전력기기 / 원전 / AI데이터센터

진입 전 가격: 91,300
상한가 가격:  91,500
일중 고점 도달: 11:23 → 풀렸다가 재진입

오늘 거래대금: X,XXX억 (현재 12위)

Historical (Layer 2, n=4):
P(갭상)=100% / E[갭]=+7.8%

→ 박민준 패턴: 상한가 진입 직후 매수
→ 상세는 14:50 결정 레포트
```

길이는 짧게. 텔레그램 푸시로 휴대폰 lock screen에서 즉시 인지 가능해야 함.

## 텔레그램 메시지 포맷 가이드

### 길이 제한

- 텔레그램 1메시지 = 4096자 제한
- 결정 레포트는 길 수 있음 → 종목별로 분할 발송 또는 핵심만

### 마크다운 적용

- `*굵게*` → 굵은 글씨
- `_기울임_` → 기울임
- 코드블록은 backtick 3개 (등폭 폰트로 표 정렬에 유용)

### 메시지 prefix 규칙

| Prefix | 의미 | 색깔감 |
|---|---|---|
| 📊 [모닝] | 정기 | 회색 |
| 📊 [추적-HH:MM] | 정기 추적 | 회색 |
| 🎯 [결정-14:50] | 결정 | 파랑 |
| 🚨 [상한가] | 이벤트 | 빨강 |
| 📝 [사후] | 사후 | 회색 (이메일) |
| ⚠️ [에러] | 시스템 장애 | 빨강 |

---

## 표 정렬 가이드

텔레그램 등폭 폰트(코드블록 안)에서 표 정렬:

```
종목         P(갭상)  E[갭]   Kelly%
제룡전기     100%    +8.9%   20.0%
대한전선      80%    +5.2%   12.5%
```

- 고정폭 컬럼 + 왼쪽 정렬
- 너무 길면 줄바꿈 → 모바일 가독성 우선

---

## 다국어 / 통화 단위

- 모든 시각: KST (Asia/Seoul)
- 통화: 원 (KRW)
- 큰 금액 표기: 억 단위 (1,234,567,890원 → 12.3억)
- 종목코드: 6자리 (앞에 0 채움)
- 등락률: 소수점 2자리 (+5.23%)

---

## 보안 / 프라이버시

- 본인 매수 정보는 레포트에 평단/수량 표시 X (필요시 포함하더라도 텔레그램만)
- 이메일 레포트는 매수 정보 제외
- API 키, 텔레그램 토큰 절대 레포트에 노출 X
