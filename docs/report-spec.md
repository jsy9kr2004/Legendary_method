# 레포트 명세 (report-spec.md)

종배 매매 의사결정을 보조하는 레포트 포맷 정의.

## 레포트 종류

| 시점 | 종류 | 채널 | 길이 |
|---|---|---|---|
| **`/on` ~ `/off`** | **실시간 모니터링** (M6, 24h 토글) ★ | 텔레그램 (메시지 편집) | 종목당 ~10줄, 1~2초 갱신. 평일 09:00 자동 ON, `/off` 로만 종료 (round 18) |
| 09:30 | 모닝 정기 | 텔레그램 | 중간 |
| 11:00 | 1차 추적 | 텔레그램 | 짧음 |
| 13:00 | 2차 추적 | 텔레그램 | 짧음 |
| 14:00 | 3차 추적 | 텔레그램 | 짧음 |
| **14:50** | **결정** ★ | **텔레그램** | **상세** |
| 상한가 진입 | 이벤트 트리거 ★ | 텔레그램 푸시 | 매우 짧음 |
| 16:00 | 사후 정기 | 텔레그램 | 매우 상세 (4096자 초과 시 자동 분할) |

## 1. 모닝 정기 레포트 (09:30)

목적: 시장 국면 판단 + 어제 매수 종목 갭 결과 확인.

```markdown
📊 [모닝] 2025-05-05 (월)
═══════════════════════════════

[시장 국면 지표]
KOSPI 시초:    2,XXX.XX  (전일 +X.XX%)
200일 이평:    2,XXX.XX  → 시초 위/아래
60일 수익률:   +X.XX%
VKOSPI:        XX.XX
20일 음봉비율: XX%

야간 미증시: S&P +X.XX% / NASDAQ +X.XX%
시간외 단일가 흐름: 강세/약세 (종목 수 기준)

[보유 종목 갭 결과]  ← Zeta 본인 매수 종목 (수동 입력)
제룡전기  매수가 91,300 → 시가 9X,XXX (+X.X%)
                          → 권장: 시초 익절

→ 대세상승장 판단: [Zeta 직관]
```

## 4.5 실시간 모니터링 (M6) — 24h 사용자 토글 (round 18)

운영: 봇 명령 polling 24h 상시. 평일 09:00 자동 ON, `/off` 로만 종료. 사용자가 임의 시점에 `/on`/`/off` 가능.

목적: 주도주 부상/교체/자금이탈을 1~2초 단위로 감지. **메시지 편집(`editMessageText`)** 으로 종목당 메시지 1개를 계속 갱신해서 푸시 알림은 신규 종목 진입 시점에만 1회.

### 종목별 모니터링 메시지 (편집 갱신)

종목은 두 가지 모드 중 하나. `/buy` 명령으로 감시 → 보유 전환, `/sell` 로 복귀.

### 알림 메시지 정책 (정정 round 17 — 카드 외 푸시 폐기)

**M6 모니터링 카드 외 별도 푸시 알림은 모두 폐기.** TRANSITION/GRACE/강한 부상/자금 이탈/AVOID/R15 매도 트리거 발화는 모두 위 카드 안의 색상·이모지·사유 한 줄로 통합 표시. `editMessageText` 1~2초 in-place 갱신만.

근거: 사용자가 평일 09:00~10:30 텔레그램 채팅을 띄워놓고 카드를 직접 보는 워크플로우. 카드를 갱신하면서 동일 정보를 푸시로 또 보내면 중복 + 폭주. 갱신만으로 사람이 빠르게 인지 가능.

푸시 ON 메시지가 발생하는 경우는 **M6 외부 이벤트**만:
- 🚨 [상한가] 모니터링 안 하던 종목 포함 (`src/report/event.py`)
- ⭐ 자동 주도주 신규 추가 — 카드가 아직 없는 종목의 첫 발송 1회 (이후엔 갱신만, 푸시 X)
- 14:50 결정 레포트, 16:00 사후 레포트 등 정기

**메시지 N분마다 삭제/재발송 패턴 X.** 채팅이 다른 알림으로 밀려 안 보일 일이 없으니 (카드 외 push 폐기 → 카드 메시지 1개만 유지), 메시지 sliding 패턴은 도입하지 않음.

## 5. 사후 정기 레포트 (16:00)

목적: 오늘 시그널 리뷰 + 시간외 단일가 흐름 + 다음날 갭 예측.

**텔레그램으로 발송** (마크다운 형식). 4096자 초과 시 `send_message`가 자동 분할.

```markdown
📝 [사후] 2025-05-05 종배 일일 리뷰
═══════════════════════════════

[오늘의 종배 후보 (재집계)]
... (14:50 레포트 + 종가 확정 정보)

[시간외 단일가 동향 16:00~]
제룡전기: 91,500 → 9X,XXX (+X.X%)  → 다음날 갭상 예고
```

## 텔레그램 메시지 포맷 가이드

### 길이 제한

- 텔레그램 1메시지 = 4096자 제한
- 결정 레포트는 길 수 있음 → 종목별로 분할 발송 또는 핵심만

### 마크다운 적용

- `*굵게*` → 굵은 글씨
- `_기울임_` → 기울임
- 코드블록은 backtick 3개 (등폭 폰트로 표 정렬에 유용)

### 메시지 prefix 규칙

| Prefix | 의미 | 색깔감 |
|---|---|---|
| 📊 [모닝] | 정기 | 회색 |
| 📊 [추적-HH:MM] | 정기 추적 | 회색 |
| 🎯 [결정-14:50] | 결정 | 파랑 |
| 🚨 [상한가] | 이벤트 | 빨강 |
| 📝 [사후] | 사후 | 회색 (이메일) |
| ⚠️ [에러] | 시스템 장애 | 빨강 |
| [모니터링#N] | 실시간 모니터링 (편집) | 무알림 |

(정정 round 17: 🔥/🔄/⚠️/⚡/🆕/🟢/🔴/🛑 prefix 모니터링 알림은 모두 폐기. 카드 안 색상·이모지로 통합.)

## 표 정렬 가이드

텔레그램 등폭 폰트(코드블록 안)에서 표 정렬:

```
종목         P(갭상)  E[갭]   Kelly%
제룡전기     100%    +8.9%   20.0%
대한전선      80%    +5.2%   12.5%
```

- 고정폭 컬럼 + 왼쪽 정렬
- 너무 길면 줄바꿈 → 모바일 가독성 우선

## 다국어 / 통화 단위

- 모든 시각: KST (Asia/Seoul)
- 통화: 원 (KRW)
- 큰 금액 표기: 억 단위 (1,234,567,890원 → 12.3억)
- 종목코드: 6자리 (앞에 0 채움)
- 등락률: 소수점 2자리 (+5.23%)
