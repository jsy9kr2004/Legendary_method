# 실시간 모니터링 가이드 (초보자용)

이 문서는 텔레그램 봇이 보여주는 카드를 처음 보는 사람이 **30분 안에 카드를 읽고
판단할 수 있도록** 만드는 것이 목표입니다. 주식 용어를 처음 듣는 사람도 이해할
수 있게 풀어 썼습니다.

대상 시스템: M6 실시간 모니터링 워커 (`src/dashboard/worker.py`).
관련 룰 문서: 더 깊은 정의는 [`docs/jongbae-strategy.md`](jongbae-strategy.md).

---

## 1. 이 시스템이 뭘 하는가

한 줄: **"단타 매수 후보 + 보유 종목 청산 시그널을 1~3초 단위로 텔레그램 카드에
계속 갱신해서 보여준다."**

봇이 절대 안 하는 것:
- 자동 매수 / 자동 매도 (사용자 정책: CLAUDE.md "자동 매매 절대 금지")
- 추천 ("이 종목 사세요" X — 점수와 시그널만 보여줌, 판단은 사람)
- 사용자 계좌 접근

사용자가 하는 것:
- 텔레그램 채팅창을 열어두고 카드 변화를 직접 본다
- 매수하면 `/buy 091340` 입력 (가격은 자동 보충)
- 청산하면 HTS/MTS 에서 직접 매도 후 `/sell 091340`

## 2. 카드 한 장 읽기

봇은 종목 1개당 텔레그램 메시지 1개를 만들어, 같은 메시지를 1~3초마다 내용만
바꿉니다 (`editMessageText`). 푸시 알림 X → 휴대폰이 안 울림. 채팅창을 켜두고
직접 봐야 합니다.

카드는 3가지 모드:

| 모드 | 언제 뜨나 | 헤더 prefix |
|---|---|---|
| **주도주 (AUTO)** | 봇이 자동으로 잡은 "오늘의 단타 1군" | `[모니터링] ... ⭐자동/주도주` |
| **부상 후보 (RISING)** | 매수 점수가 높아진 후보 | `[부상] ... ⚡부상 후보` |
| **수동 (MANUAL)** | 사용자가 직접 추가한 종목 | `[모니터링] ... 🔵수동` |
| **보유 (HOLD)** | `/buy` 친 보유 종목 — 청산 시그널 표시 | `[보유]` |

### 2.1 감시 카드 예시 (주도주)

```
[모니터링] 대한광통신 (091340) ⭐자동/주도주  🟢 STRONG +6.5점
테마: AI데이터센터 / 광케이블
사유: +1 거래대금 50위내 / +2 가속 동반 (5m 5.0 / 1m 5.0) / +2 장대양봉 (윗꼬리 9%)
10:23:45  91,300원 (+30.0%) 🔴상한가
+29% 매도가: 90,500원 (현재가 대비 -0.9%)
거래대금: 1247억 (1위)  회전율: +18.3%
🟢 5분봉가속: ↑ 5.5배 (자금 유입 가속)  5분합 50억
🟢⚡ 1분봉가속: ↑ 5.5배 (1분봉 부상)  최근1분 10억
🟢 체결강도: 142 (매수 우세)  5MA 138  1MA 135  매수비율 +60.0%
🟢 호가: 매수 320,000 / 매도 45,000 (7.1배)
1호가: 매수 91,200원 (850) / 매도 91,300원 (120)
```

라인별 의미:
- **헤더** — 종목 + 등급 색깔/점수. STRONG 이면 매수 검토 가능 (사람이 결정)
- **테마** — 이 종목이 속한 단타 테마 (네이버 금융 기준)
- **사유** — 점수 가산/감점 항목 상위 3개
- **시각 / 가격 / 전일 대비** — 현재 KIS 가격 + 전일 종가 대비 등락률 + 상한가면 🔴
- **+29% 매도가** — 상한가 +30% 직전에서 매매 정지 회피용 매도 호가. 보유 시 참고
- **거래대금 / 순위 / 회전율** — 코스피+코스닥 전체 중 거래대금 몇 위, 회전율은 시총 대비 거래량
- **5분봉가속 / 1분봉가속** — 자금이 얼마나 빨리 들어오고 있는지 (자세한 의미는 4장)
- **체결강도** — 적극 매수 vs 적극 매도 비율 (100 = 균형)
- **호가** — 호가창 매수/매도 잔량 비율 (참고용, 메인 X)

### 2.2 부상 후보 카드 (RISING)

```
[부상] 계양전기 (012200) ⚡부상 후보  🟡 WATCH +3.5점
테마: AI
사유: +1 거래대금 50위내 / +2 장대양봉 (윗꼬리 12%) / +0.5 호가 2.5배 (보조)
10:23:45  8,920원 (+14.4%)
+29% 매도가: 10,060원 (현재가 대비 +12.8%)
거래대금: 450억 (8위)  회전율: +11.2%
🟢 5분봉가속: ↑ 2.8배 (유입)  5분합 20억
🟢 1분봉가속: ↑ 2.0배 (유입)  최근1분 4억
🟡 체결강도: 118 (균형)  5MA 115  1MA 120  매수비율 +54.0%
🟢 호가: 매수 180,000 / 매도 72,000 (2.5배)
1호가: 매수 8,910원 (1,200) / 매도 8,920원 (800)
매매 결정 시 → /add 012200
```

차이점:
- prefix 가 `[부상]` 으로 다름
- "매매 결정 시 → `/add 012200`" 안내가 맨 아래
- WATCH 등급(2점 이상)은 통과한 것 — 즉 봇이 "볼 만하다" 판단한 후보

### 2.3 보유 카드 (`/buy` 한 뒤)

```
[보유] 대한광통신 (091340)  🟢 STRONG +6.5점
테마: AI데이터센터 / 광케이블
사유: +1 거래대금 50위내 / +2 가속 동반 / +2 장대양봉
10:23:45 (+1,200초)  🔴92,500원 (+31.8%) / 🔴91,300(+1.3%)
거래대금: 1350억 (1위)  회전율: +19.0%
🟢 5분봉가속: ↑ 4.8배 (자금 유입 가속)  5분합 45억
🟢⚡ 1분봉가속: ↑ 3.5배 (1분봉 부상)  최근1분 9억
🟢 체결강도: 138 (매수 우세)  5MA 135  1MA 123  매수비율 +58.0%
🟢 호가: 매수 280,000 / 매도 55,000 (5.1배)
1호가: 매수 92,400원 (650) / 매도 92,500원 (180)
─ 청산 시그널 ─
❌ 체결강도 5MA 100 하향 (현재 5MA 135 / 1MA 123)
❌ Bearish Divergence (가격 +0.80% / 체결강도 -3)
❌ 자금 고갈 (1분 가속 < 0.5) — 현재 3.5배
❌ 윗꼬리 50%↑ 음봉 (1분봉 기준)
❌ VI 발동 후 5분 내 재상승 실패
```

추가/변경된 라인:
- **`10:23:45 (+1,200초)`** — 매수 후 경과 시간(초). 매 tick 갱신
- **`🔴92,500원 (+31.8%) / 🔴91,300(+1.3%)`** — `현재가(오늘 등락) / 매수가(매수 후 손익)`
- **─ 청산 시그널 ─** — 5개 시그널. ❌ = 미발화, ✅ = 발화. 한 개라도 ✅ 되면 청산 검토

손절선/익절선은 화면에 표시 안 함 — 매수가 91,300원에서 -1.5% = 89,930원, +2% = 93,126원
같은 건 머릿속으로 계산 가능 (또는 HTS 에 알람 설정).

## 3. 후보 선정 과정 — 4단계 깔때기

봇은 코스피+코스닥 약 2,700종목 중 매 3초마다 "지금 살 만한 종목"을 추립니다.
한국투자증권 API 호출 한도가 초당 20개라 모든 종목을 다 보지 못합니다. **단계별로
값싼 필터부터 통과시켜서 비싼 호출은 통과한 종목에만** 합니다.

### Stage 0 — 무료 필터 (이미 받은 데이터만 사용)

거래대금 50위 종목 리스트를 가져옴. 그 중에서:
- **ETF / ETN / 리츠 / 스팩 / 펀드 제외** — 단타 대상 아님 (KODEX, TIGER 등 자동 차단)
- **양봉만** — 전일 종가 대비 0% 초과 (음봉은 인버스 매매 아니므로 제외)
- **상한가 임박 제외** — 일중 +29% 넘으면 매매 정지 임박이라 제외

→ 약 20~30종목 남음

### Stage 1 — 회전율 상위 15 (여전히 무료)

남은 종목 중에서 **회전율(거래대금 ÷ 시가총액)** 상위 15개만 남김.

왜 회전율? **거래대금 절대값으로 보면 항상 삼성전자/하이닉스가 1위**입니다.
하지만 단타로 노리는 종목은 시총이 작아도 거래량이 폭증한 종목이라, 시총 대비
거래량 비율(=회전율)이 단타 자금 유입의 진짜 강도입니다.

→ 15종목 남음. 여기까지 KIS API 호출 0번

### Stage 2 — 모멘텀 필터 (1분봉 분석, 종목당 1콜)

15종목 각각의 분봉 시세를 받아서:
- **5분봉 거래대금 가속 > 0.8** — 자금이 아직 들어오고 있는지 확인
- **약한 봉 패턴 아님** — 윗꼬리가 크거나 음봉으로 끝났으면 매수세 약함 → 제외

흥아해운처럼 거래대금은 1위인데 모멘텀이 죽은 종목은 여기서 걸러집니다.

→ 약 5~10종목 남음

### Stage 3 — 체결강도 필터 (체결 비율 분석, 종목당 1콜)

각 종목의 **체결강도(VP)**를 확인. 100 미만이면 제외.

체결강도 < 100 = 능동적 매도가 능동적 매수보다 많은 상태. 진입 시 즉시 손실 시작.

→ 약 3~5종목 남음

### Stage 4 — R14 매수 점수 (호가 + 외국인 + 종합 점수, 종목당 2콜)

남은 종목에 호가/투자자 데이터까지 받아서 **종합 매수 점수(R14)** 계산. 점수가
2.0(WATCH) 이상이어야 부상 후보 카드로 노출.

→ 최종 0~5개가 부상 후보 카드로 표시

### 비용 시뮬레이션

- 3초 tick 당 평균 약 33 API 호출 (KIS 한도 60의 55%)
- 흥아해운류는 Stage 2 에서 죽음 → 비싼 호출 절감
- VP 약한 종목도 Stage 3 에서 죽음

### 흥아해운 회귀 케이스

> "거래대금 1위 + 회전율 19.4% + 호가 매수가 매도의 5.3배"

거래대금만 보면 매수 후보. 하지만:
- 5분봉 가속 0.8배 (자금 식어감) → Stage 2 탈락
- 직전 1분봉이 윗꼬리 큰 음봉 → Stage 2 탈락

→ **이 종목은 부상 후보 카드로 안 뜸**. 가짜 매수 신호 차단 (`tests/test_dashboard_worker.py::test_rising_funnel_filters_heunga_haewoon` 회귀).

## 4. 용어 사전 (한국 단타 기초)

### 4.1 봉 차트 기초

**봉(candle)**: 시세를 시각화한 막대. 시가/고가/저가/종가 4개로 그림.

```
        ┃ ← 위꼬리 (윗그림자) — 고가 ~ 종가(또는 시가) 사이
       ┃▓┃ ← 몸통 — 시가 ~ 종가 사이
       ┃▓┃     ▓이면 양봉 (종가>시가)
       ┃▓┃     ░이면 음봉 (종가<시가)
        ┃ ← 아래꼬리 — 저가 ~ 시가(또는 종가) 사이
```

**양봉 (bullish)**: 종가가 시가보다 높음. 매수 우세
**음봉 (bearish)**: 종가가 시가보다 낮음. 매도 우세
**도지 (doji)**: 종가 ≈ 시가. 균형/관망

**5분봉 / 1분봉**: 봉 1개가 몇 분간의 거래를 모은 건지. 단타는 보통 1분봉/5분봉
같이 봅니다. 짧을수록 노이즈 많음.

**윗꼬리 50%↑ 음봉**: 봉 길이의 절반 이상이 위쪽 그림자고 종가가 시가보다 낮음.
→ 고점 찍고 빠진 모양. **강력한 매도 신호**.

```
예시 (윗꼬리 큰 음봉):
        ┃
        ┃  ← 위꼬리 (전체 60%)
        ┃
       ┃░┃ ← 몸통 음봉 (전체 30%)
        ┃  ← 아래꼬리 (전체 10%)
```

### 4.2 거래대금 / 회전율

**거래대금**: 그날 그 종목에서 발생한 매매 총액(원). 시장 전체에서 1~50위까지가
단타 후보군.

**시가총액**: 발행 주식수 × 현재가. 회사 규모.

**회전율 (turnover)**: 거래대금 ÷ 시가총액. **시총 정규화 거래량**. 단위 %.

왜 회전율을 보나:
- 거래대금 1위 = 항상 삼성전자/하이닉스 (시총 큰 종목). 단타 대상 아님
- 시총 작은데 거래대금이 시총의 10~30% = 회전율 10~30% = **단타 자금 폭증 상태**
- 한국 단타 통설: **회전율 1위가 진짜 주도주**

### 4.3 거래대금 가속 (vol_accel)

자금이 들어오는 속도가 빨라지는지 보는 지표. 직전 평균 대비 몇 배인지.

**1분봉 가속**: 최근 1분 거래대금 ÷ 직전 5분 평균 분당 거래대금
- 1.0 = 평소대로
- 3.0 = 평소의 3배 (자금 가속 진입)
- 5.0+ = 매우 빠른 진입 → 1분봉 부상 ⚡
- 0.5 미만 = 자금 빠짐 → 자금 고갈

**5분봉 가속**: 최근 5분 거래대금 ÷ 직전 20분 평균 5분 거래대금
- 같은 개념이지만 윈도우가 길어 노이즈 적음
- 10배+ = 강한 부상 ⚡
- 0.6 미만 = 자금 이탈 ⚠

### 4.4 상한가 / +29%

한국 주식은 하루에 **±30%** 까지만 변동 가능. 상한가 = +30% 도달.

**+29% 룰**: 봇은 +29% 도달하면 부상 후보에서 자동 제외. 이유:
- +29~30% 사이는 상한가 임박 → 매매 정지 또는 매도 호가가 비어있음
- 거기서 매수 시도해도 체결 안 됨
- "곧 상한가" 가 매수 진입의 정설이고, "이미 상한가" 는 너무 늦음

### 4.5 체결강도 (VP, Volume Power)

**계산**: `VP = 능동 매수 체결량 / 능동 매도 체결량 × 100`

- VP 100 = 균형
- VP > 110 = 매수 우세 (적극 매수가 적극 매도보다 10%↑ 많음)
- VP > 120 = 매수 강세 🟢
- VP < 80 = 매도 우세 🔴

**왜 호가 잔량이 아니라 체결강도?**
- 호가 잔량 비율은 **스푸핑** 취약 (큰 매수 호가 띄워놓고 가격 올라가면 취소)
- 매도자가 시장가로 던지면 매도 잔량이 줄어 비율만 크게 보임 → 가짜 신호
- 체결강도는 **실제 체결된 거래**라 조작 불가

**VP_5MA / VP_1MA**: 체결강도의 5분 / 1분 이동평균
- 즉시값(VP)은 흔들림 큼 → 평균 내서 추세 봄
- 5MA 가 안정적, 1MA 는 빠른 반응 (참고용)

### 4.6 다이버전스 (Divergence)

가격과 체결강도가 따로 노는 현상.

- **Bearish Divergence**: 가격은 오르는데 체결강도는 떨어짐
  - "겉으로는 상승, 속으로는 약화" → 고점 임박 신호
  - 5분간 변화 비교
- **Bullish Divergence**: 가격은 내리는데 체결강도는 오름
  - "표면 하락, 매수세 매집 중" → 매수 매집 신호

### 4.7 VI (Volatility Interruption, 변동성 완화 장치)

급격한 가격 변동 시 KRX 가 강제로 **2분간 거래 정지**.
- 동적 VI: 직전 체결가 대비 ±2~3% 이상 변동 시 발동 (호가 단위)
- 정적 VI: 시가 대비 ±10% 변동 시 발동

**VI 후 재상승 실패**: VI 풀린 뒤 5분 안에 직전 고가 회복 못 하면 → 청산 시그널.
원래 상승 가속이 강했는데 VI 라는 외부 충격 후 회복 못 하면 모멘텀이 꺾인 것.

## 5. 매수 점수 (R14) — 어떻게 매기나

부상 후보 카드의 **사유** 라인에 표시되는 점수.

| 조건 | 점수 |
|---|---|
| 거래대금 회전율 상위 (10위 안) | +1 |
| 5분봉 가속 + 1분봉 가속 둘 다 강함 | +2 |
| 5분봉 가속 + 1분봉 가속 둘 다 약함 | -3 |
| 깨끗한 양봉 (윗꼬리 30% 미만) | +2 |
| 약한 봉 (음봉 또는 윗꼬리 40%↑) | -2 |
| 체결강도 > 110 + 5MA > 100 | +2 |
| 체결강도 < 100 | -2 |
| 1분 가속 매우 강함 (vol_accel_1m > 5) | +1 |
| 자금 고갈 (vol_accel_1m < 0.5) | -1 |
| Bullish Divergence | +2 |
| Bearish Divergence | -2 |
| 호가 잔량 매수/매도 > 3 (보조) | +0.5 |

**등급**:
- 🟢 STRONG (≥ 5점) — 강한 매수 신호 (사람이 최종 결정)
- 🟡 WATCH (≥ 2점) — 감시할 가치 있음
- ⚫ NEUTRAL (≥ -1점) — 보통
- 🔴 AVOID (< -1점) — 매수 회피

**중요**: 이 점수는 **권유가 아닙니다**. 단지 한국 단타 통설을 가중합으로 정리한
참고 수치. 점수만 보고 매수 결정 X — 사용자가 차트도 보고 호가창도 보고 판단해야
합니다.

흥아해운 시나리오 점수: -3 ~ -5 → 🔴 AVOID (자동 거름)

## 6. 매도 시그널 (R15) — 보유 후 청산 판단

`/buy` 한 뒤 보유 카드의 **─ 청산 시그널 ─** 섹션. 5가지가 ❌(미발화) / ✅(발화)
로 매 tick 갱신.

### 5가지 청산 시그널 (C1~C5)

| 시그널 | 발화 조건 | 의미 |
|---|---|---|
| **C1 체결강도 5MA 100 하향** | VP_5MA 가 100 위에서 → 100 아래로 떨어진 순간 | 매수세가 평균적으로 약화 확정 |
| **C2 Bearish Divergence** | 가격↑ + VP_5MA↓ (5분 변화) | 표면 상승, 속으로 매수세 빠짐 — 고점 신호 |
| **C3 자금 고갈** | 1분 가속 < 0.5 가 2분 지속 | 자금이 빠지는 게 확정적 |
| **C4 윗꼬리 50%↑ 음봉** | 1분봉이 윗꼬리 50% 이상 + 음봉 | 고점 찍고 빠지는 봉 출현 |
| **C5 VI 후 재상승 실패** | VI 발동 후 5분 내 고점 회복 X | 모멘텀 꺾임 |

### 가격 기준 청산 (A/B 그룹)

카드에 별도 표시하지 않지만 사용자가 직접 인지해야 함:

**A 그룹 — 손절 (최우선, 무조건)**
- A1: 매수가 대비 **-1.5%** 이탈 (예: 91,300 → 89,930)
- A2: 진입한 봉의 저점 이탈
- A3: 5분봉 종가 기준 5분 이평선 이탈
- A4: 진입 후 N분 (기본 10분) 내 +0.5% 못 가면 청산

**B 그룹 — 익절 (계획적, 분할)**
- B1: 매수가 대비 **+2.0%** 도달 → 1/3 익절 (원금 회수 모드)
- B2: 매수가 대비 **+3.5%** 도달 → 1/3 익절
- B3: 나머지 1/3 → 트레일링 스탑 (고점 대비 -1.5%)

**왜 카드에 안 그리나?**
- 카드 라인 절약 — 매수가가 위 가격 라인에 있으니 사용자가 머릿속으로 계산 가능
- HTS/MTS 에서 가격 알람 설정하는 게 더 정확 (1초 단위)

### 매수와 매도의 차이

| 항목 | 매수 (R14) | 매도 (R15) |
|---|---|---|
| 논리 | AND (다수결 통과) | OR (하나만 발동) |
| 임계값 | 보수적 (STRONG 5점 이상) | 공격적 (한 시그널만 떠도 검토) |
| 반응 | 분 단위 | 초 단위 (특히 손절) |
| 실패 시 | 기회비용 | 실현 손실 |
| 자동화 | 영구 미지원 (사용자 결정) | 영구 미지원 (사용자 결정) |

**철학**: 진입은 안 들어가면 그만이지만, 청산은 못 나오면 손실 실현. 그래서 매도
시그널은 더 빠르고 다양하게 봅니다.

## 7. 봇 명령 사용법

봇 명령은 24시간 받음 (평일/주말/장 시간 외 무관).

### 모니터링 ON/OFF
- `/on` 또는 `/start` — 카드 갱신 시작
- `/off` 또는 `/pause` — 카드 갱신 중지 (카드 메시지 정리)
- 평일 09:00 자동 ON, 10:30 자동 OFF 폐지 — `/off` 로만 종료

### 종목 추가/제거
- `091340` (6자리 숫자만 보내기) — 감시 모드 추가/해제 토글
- `/list` — 현재 모니터링 중인 종목 목록
- `/clear` — 수동 추가분만 일괄 해제

### 매수/매도 (보유 모드 등록)
- `/buy 091340` — 보유 모드 진입. **가격은 봇이 KIS 시세에서 자동 보충**
- `/buy 091340 91300` — 매수가 명시 (HTS 슬리피지 큰 경우 등)
- `/buy 091340 91300 5` — 매수가 + 시간 손절 5분 오버라이드
- `/sell 091340` — 보유 모드 해제 (감시 모드 복귀)

### 기타
- `/status 091340` — 해당 종목 카드 강제 재발송

## 8. 흔히 빠지는 함정

- **거래대금 1위 = 주도주 X** — 항상 대형주. 회전율 1위가 진짜 단타 주도주
- **호가 잔량 비율만으로 매수 X** — 허매수/스푸핑 함정 (흥아해운 케이스)
- **상한가 도달 후 추격 매수 X** — 봇이 +29% 룰로 자동 거름. 따라가도 +30% 막힘
- **점수가 STRONG 이라고 무조건 매수 X** — 차트/호가창/시장 분위기 같이 봐야
- **매도 시그널 5개 다 ❌ 라고 안심 X** — 가격 기반 손절선(-1.5%)이 우선
- **카드 외 푸시 거의 없음** — 채팅창 안 보면 모름. `/on` 후 항상 띄워둘 것
